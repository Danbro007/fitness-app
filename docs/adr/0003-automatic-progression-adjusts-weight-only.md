# ADR-0003：首版自动渐进只调整重量

- 状态：已接受
- 日期：2026-07-22

## 背景

计划动作的训练目标由组数、每组次数和重量组成。早期设想允许系统同时建议调整三项，但多个变量同时变化会放大训练总量变化，也会让用户难以理解系统为何调整。

## 决策

首版自动渐进只调整计划动作的目标重量。

系统继续读取实际组数、实际次数、实际重量和主观感受，用于判断是否应增加、保持或降低重量。系统不得自动修改目标组数和每组目标次数；用户可以在计划草稿中手动修改它们。

没有历史记录的新动作可以由系统给出初始组数和次数建议，用户确认后成为基线。这属于首次建档，不属于后续自动渐进。

重量调整必须遵守以下约束：

- 同一动作至少存在最近 3 次已完成训练；
- 趋势按同一动作判断，不按器械合并；
- 每次最多调整一个当前训练场地可用的重量档位；
- 结果先进入草稿，用户确认后才写入正式计划。

## 影响

- 自动调整规则和 AI 输出契约不得产生组数、次数自动变更。
- 草稿应解释用于判断的组数、次数、重量和感受，同时突出唯一自动变化项是重量。
- 将来若要自动调整组数或次数，需要重新评估训练总量、安全边界和交互解释，并新增决策记录。
