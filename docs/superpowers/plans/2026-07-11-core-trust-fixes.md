# i fitness 核心信任修复

## 范围

- 首次打开必须先完成本地训练档案，不展示未校准的训练重量。
- 提前结束且未完成目标组数时，保存为“部分完成”，不计入本周完成次数。
- 手动饮食记录在软键盘弹出时仍可滚动至保存操作，并为数值字段启用数字键盘。

## 验收

- [x] 新数据库启动时进入首次训练设置；保存后才进入五项主导航。
- [x] `1/7` 组提前结束后，训练总结显示“部分完成”，计划恢复为可再次开始，周完成数不增加。
- [x] 全部目标组完成后结束，仍显示“训练完成”并增加周完成数。
- [x] 手动餐食表单带软键盘时可滚动至保存操作。
- [x] 相关 JVM / 设备测试、Debug APK 构建和模拟器关键路径通过。
